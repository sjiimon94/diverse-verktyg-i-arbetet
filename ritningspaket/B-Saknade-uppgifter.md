# B. SAKNADE UPPGIFTER SOM MÅSTE BEKRÄFTAS

Följande uppgifter saknas för att kunna göra handlingen 100 % produktionsklar.
Varje punkt måste bekräftas innan bygghandlingsstatus kan uppnås.

## Grund och mark

- [ ] Exakt grundtyp (plintgrund, kantbalk, platta på mark, krypgrund?)
- [ ] Markförhållanden och geotekniska förutsättningar
- [ ] Befintlig grundnivå på huvudbyggnaden
- [ ] Frostfritt djup och isoleringsåtgärder i grund
- [ ] Radonskyddsåtgärder (om aktuellt)

## Geometri och mått

- [ ] Exakt slutgiltig plangeometri med verifierade yttermått
- [ ] Kontrollmätning av befintlig husvägg (längd, höjd, planhet)
- [ ] Exakt stomhöjd, bekräftat med uppmätning
- [ ] Exakt bredd på den förstorade öppningen (markerad med blå streck i skiss)
- [ ] Exakt mått och funktion för den 45 mm tjocka anslutningsdelen mot befintligt hus
- [ ] Nivåskillnad befintligt golv – ny FG +68.12 (om någon)

## Öppningar

- [ ] Exakta öppningsmått för samtliga dörrar (karm- och hålmått)
- [ ] Exakta öppningsmått för samtliga fönster (karm- och hålmått)
- [ ] Typ och fabrikat dörrar och fönster
- [ ] Bröstningshöjder fönster
- [ ] Tröskelhöjder dörrar
- [ ] Antal och placering av samtliga öppningar (bekräftat)

## Anslutning mot befintligt hus

- [ ] Exakt konstruktion befintlig yttervägg (lecasten – tjocklek, typ)
- [ ] Anslutningsdetalj hög takpunkt – förbandsprincip
- [ ] Infästningstyp i befintlig vägg (kemankare, genomgående bult, vinklar)
- [ ] Tätningsprincip (fogmassa, plåtbeslag, membran)
- [ ] Behov av håltagning i befintligt tak eller vägg
- [ ] Befintligt taks överkant och underkant vid anslutningspunkt

## Taktäckning och avvattning

- [ ] Taktäckningsmaterial (papp, duk, plåt, takpannor?)
- [ ] Undertaksprincip (underlagspapp, underlagsduk, luftat undertak?)
- [ ] Takavvattning (hängränna, invändig takbrunn?)
- [ ] Snörasskydd
- [ ] Tillgänglighet för takarbete (taklucka, takstege?)

## Brand, energi och fukt

- [ ] Brandklass krav (EI30, EI60?)
- [ ] U-värdesberäkning för tillbyggnaden
- [ ] Energiberäkning jämförelse med befintlig byggnad
- [ ] Fuktsäkerhetskontroll och fuktsäkerhetsprojektering
- [ ] Ventilationslösning / tilluft / frånluft

## Installationer

- [ ] El-matning, centralplacering
- [ ] VVS-dragning (vatten, avlopp, värme)
- [ ] Ventilationskanaler och aggregatplacering
- [ ] Eventuella golvvärmerör i ny platta/golvbjälklag

## Myndighetskrav

- [ ] Bygglov / anmälan (beviljat? villkor?)
- [ ] Eventuella planbestämmelser och detaljplan
- [ ] Tillgänglighetskrav (rullstolsanpassning, trösklar)

## Övrigt

- [ ] Markarbeten och dränering
- [ ] Ytskikt invändigt (golv, väggar, tak per rum)
- [ ] Beställarens namn och kontaktuppgifter
- [ ] Projektnummer

---
*Samtliga ovanstående punkter ska besvaras innan handlingen går från PRELIMINÄR till BYGGHANDLING.*
